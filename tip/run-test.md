## Run All test
```sh
cd test/
julia  -t auto -p auto --check-bounds=yes --startup-file=no --depwarn=error ./runtests.jl  
```

```sh
cd test/
julia  -t auto -p auto --check-bounds=yes --startup-file=no --depwarn=error ./runtests.jl  Libdl cmdlineargs compiler/codegen misc
```
